Event Settings

Configure your event to match your production needs.

Basic Information​

SettingDescription
TitleThe name of your event
DescriptionPublic description for your event
DatesStart and end dates
TimezoneThe event's local timezone
Message of the Day (MOTD)Displayed to all participants

Submission Settings​

Control how performers submit acts to your event:

SettingOptions
Accept SubmissionsNone, Company Only, or Public
Submission DeadlineWhen submissions close
note

If no deadline is set, the first show's date is used as the default deadline.

Event Notes​

Add private notes visible only to producers and stage managers. Use these for:

  • Production schedules
  • Venue information
  • Contact details
  • Internal reminders
