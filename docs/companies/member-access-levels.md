# Member Access Levels

TroupeIT uses a role-based system to control what members can do.

## Access Level Overview

| Level | Name | Capabilities |
|-------|------|--------------|
| 8 | **Producer** | Full access: create events, manage shows, invite members, edit company settings, edit any act |
| 4 | **Stage Manager** | Manage shows: edit cue lists, control live view, manage submissions |
| 2 | **Tech Crew** | View technical details, participate in shows |
| 0 | **Performer** | Create and manage own acts, submit to shows, view show schedules |

## Owner Role

In addition to access levels, one member is designated as the company **Owner**, who has administrative control over the company including billing.

## Changing Access Levels

To change a member's access level:

1. Go to the company's **Members** page
2. Click on the member you want to modify
3. Select the new access level
4. Click **Update**

:::note Permissions Required
Only Producers and Owners can change member access levels.
:::
