# Act Details

Provide complete information for your act so producers and tech crew have everything they need.

## Basic Information

| Field | Description |
|-------|-------------|
| **Act Name** | A title for your act (e.g., "Fever", "The Swan") |
| **Description** | What's the act about? Describe the concept |
| **Performers** | Names of all performers (if group act) |
| **Duration** | How long the act runs (MM:SS format) |
| **Tags** | Keywords to help find your act (e.g., "aerial, silks, dramatic") |

## MC Introduction

Write how you'd like to be introduced. This is what the MC will read before your act. Include:

- Your performer name
- Any relevant context
- How to pronounce unusual names
- The tone you want (dramatic, playful, etc.)

## Technical Information

| Field | Description |
|-------|-------------|
| **Sound Cue** | When should music start? (e.g., "When posed", "On entrance", "After bow") |
| **Lighting Notes** | Preferred colors, styles, blackouts, specials |
| **Prop Notes** | What props do you need and where should they be placed? |
| **Cleanup Notes** | What needs to be cleaned up after your act? |
| **Crew Notes** | Any additional information for the crew |
| **Runthrough** | Do you need a tech rehearsal? Any specific requirements? |

:::tip Stage Directions
Use standard stage directions: Stage Left, Stage Right, Upstage, Downstage. Remember: stage directions are from the performer's perspective facing the audience.
:::
