# Creating an Account

Anyone can create a free account on TroupeIT.

## Sign Up Process

1. Visit [troupeit.com](https://troupeit.com) and click **Sign Up**
2. Complete the security verification (this helps prevent spam accounts)
3. Enter your email address and choose a secure password
4. Fill in your display name and username
5. Click **Create Account**

## Password Security Tips

- Use a unique password that you don't use on other sites
- Use a mix of letters, numbers, and symbols
- Consider using a password manager

After signing up, you'll be guided through a brief onboarding wizard to set up your profile.
