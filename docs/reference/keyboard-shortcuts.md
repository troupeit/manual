# Keyboard Shortcuts

Quick reference for keyboard shortcuts in TroupeIT.

## Live View Shortcuts

| Shortcut | Action |
|----------|--------|
| `Alt + M` | Toggle chat panel |
| `↑` / `↓` | Navigate cue list |
| `Enter` | Advance to selected cue |

## General Navigation

Standard browser shortcuts work throughout the app:

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next field |
| `Shift + Tab` | Move to previous field |
| `Escape` | Close modal/dialog |

## Tips

- Most actions have clickable buttons if you prefer mouse navigation
- Keyboard shortcuts work best when the relevant panel has focus
- Mobile devices don't support keyboard shortcuts (use touch controls instead)
