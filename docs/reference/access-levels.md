# Access Level Reference

Complete reference for what each role can do in TroupeIT.

## Company Access Levels

| Level | Role | Can Do |
|-------|------|--------|
| 8 | Producer | Everything: create events, edit shows, manage members, billing, edit any act in company |
| 4 | Stage Manager | Manage shows: edit cue lists, advance cues in live view, manage submissions |
| 2 | Tech Crew | View show details, participate in live view, add notes |
| 0 | Performer | Create own acts, submit to shows, view schedules |

## Detailed Permissions by Role

### All Members

- View company information
- See event and show schedules
- Use live view during shows
- Participate in chat

### Performers (Level 0+)

- Create and edit their own acts
- Submit acts to events accepting submissions
- View submission status

### Tech Crew (Level 2+)

- View technical details in cue sheets
- Access backstage information

### Stage Managers (Level 4+)

- Edit show schedules
- Add/remove acts from shows
- Control live view cue position
- Manage submissions
- Add show notes

### Producers (Level 8)

- Create and delete events
- Create and delete shows
- Invite and remove members
- Change member access levels
- Edit any act submitted to their events
- Manage company settings

### Owners

- All Producer capabilities
- Manage subscription and billing
- Delete the company
- Transfer ownership

## Special Considerations

- Only one member can be the Owner at a time
- Ownership can be transferred to another Producer
- Removing a member doesn't delete their acts (they retain ownership)
