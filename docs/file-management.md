# File Management

TroupeIT provides a file manager for organizing your media.

## Accessing Files

- Go to **Settings** â†’ **Files**
- Or access through the act editor

## Supported File Types

### Audio
- MP3
- WAV
- AIFF
- M4A
- OGG

### Video
- MP4
- MOV
- WebM

### Images
- JPG
- PNG
- GIF
- WebP

## File Operations

From the file manager you can:

- **Upload** new files
- **Search** and filter existing files
- **Attach** files to acts
- **Delete** unused files

## Storage Limits

Storage limits depend on your subscription plan. Check your company's billing page to see current usage and limits.

:::tip Cleanup
Periodically review and delete unused files to stay within your storage limits.
:::
