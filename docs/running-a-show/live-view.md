# Live View

Live View is TroupeIT's real-time show management interface. It allows your entire team to track the show's progress simultaneously.

## Accessing Live View

1. Go to the show page
2. Click **Live View** or the "Go Live" button

## Features

| Feature | Description |
|---------|-------------|
| **Current Cue Indicator** | The current act is highlighted and visible to everyone |
| **Auto-Scroll** | View automatically follows the current cue |
| **Cue Timer** | Track elapsed time for the current item |
| **Column Filters** | Show/hide columns based on your role |
| **Act Notes** | Add personal notes to any item |
| **Chat** | Communicate with your team in real-time (Alt+M shortcut) |

## View Modes

Live View offers different display options:

- **Full Cue List**: See the entire schedule at once
- **Backstage View**: Simplified view for performers
- **Single Cue**: Focus on just the current item

## Advancing Cues

- Click on any item to make it current
- Use keyboard shortcuts for quick navigation
- All connected users see updates in real-time
