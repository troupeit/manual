# Exporting and Printing

Generate professional documents from your show schedule.

## Export Formats

| Format | Description |
|--------|-------------|
| **PDF** | Professional cue sheet for printing |
| **CSV** | Spreadsheet-compatible data export |
| **Excel (XLSX)** | Full-featured spreadsheet with formatting |

## How to Export

1. Go to the show page
2. Click **Export** or the export button
3. Select your preferred format
4. Download the generated file

:::note Paid Feature
PDF export requires a paid subscription.
:::

## Printing

For printed cue sheets:

1. Export to PDF
2. Print from your PDF viewer

Or use your browser's print function directly from the show page.

## What to Print

Consider printing copies for:
- Stage manager (full cue sheet)
- Sound tech (audio cues)
- Lighting tech (lighting notes)
- Backstage crew (act order and timing)
- Performers (simplified schedule)

## Best Practices

- Print on the day of the show to capture last-minute changes
- Use color printing to preserve note colors
- Laminate or protect sheets if working near food/drinks
