# Downloading Show Assets

Download all media files for a show in one package.

## How to Download

1. Go to the show page
2. Click **Download Assets** or the download button
3. A ZIP file is prepared containing all music and media
4. Download the ZIP file to your computer

## What's Included

The download includes:
- All music files attached to acts
- Any video or image files
- Files organized by act order

## Use Cases

Downloading assets is useful for:

- **Preparing QLab** or other playback software
- **Local backups** during the show
- **Sharing files** with tech crew who need direct access
- **Offline access** in venues with poor connectivity

:::tip Day of Show
Download assets early on show day and verify all files play correctly in your playback system.
:::
