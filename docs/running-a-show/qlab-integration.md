# QLab Integration

TroupeIT can export directly to QLab, the popular macOS audio/video playback software.

## Setup Process

1. Download the show assets from TroupeIT
2. Use the provided AppleScript to import into QLab
3. Cues are created with proper labels and organization

## AppleScripts

AppleScripts for QLab integration are available in the TroupeIT installation at:

```
public/applescripts/
```

## What Gets Imported

The QLab import creates:
- Audio cues for each act's music
- Proper cue naming matching your show schedule
- Organized cue lists

## Workflow

```
TroupeIT Show → Download Assets → Run AppleScript → QLab Workspace Ready
```

:::note macOS Only
QLab is a macOS application. This integration is only available on Mac computers.
:::
