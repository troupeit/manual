# Event Settings

Configure your event to match your production needs.

## Basic Information

| Setting | Description |
|---------|-------------|
| **Title** | The name of your event |
| **Description** | Public description for your event |
| **Dates** | Start and end dates |
| **Timezone** | The event's local timezone |
| **Message of the Day (MOTD)** | Displayed to all participants |

## Submission Settings

Control how performers submit acts to your event:

| Setting | Options |
|---------|---------|
| **Accept Submissions** | None, Company Only, or Public |
| **Submission Deadline** | When submissions close |

:::note
If no deadline is set, the first show's date is used as the default deadline.
:::

## Event Notes

Add private notes visible only to producers and stage managers. Use these for:

- Production schedules
- Venue information
- Contact details
- Internal reminders
