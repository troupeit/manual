# Notification Preferences

Configure how you want to receive notifications from TroupeIT.

## Email Notifications

Control which emails you receive:

- Event and show updates
- Submission notifications
- Company invitations

## SMS Notifications

SMS notifications require a phone number to be set in your profile.

Options include:

- Enable/disable SMS alerts
- Set quiet hours to avoid notifications during sleep

## Theme Preference

Choose your preferred visual theme:

- **System**: Follows your device settings (light/dark)
- **Light mode**: Always use light theme
- **Dark mode**: Always use dark theme
